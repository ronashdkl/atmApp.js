# ATM interface

## Features

- Fast Cash
- Withdraw amount
- Check Balance
- ToDo - Statement

### Requirements

- [npm](https://www.npmjs.com/get-npm)
- [Node.js](https://nodejs.org/en/download/)

### Install dependencies

- `npm install`

### Development

- `npm run dev` client
- `npm run build` build

Then visit `http://localhost:8080/` _- or a new browser windows popped-up already -_ to preview your website
